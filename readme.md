# CharlotteJS
## Node.js in 30 Minutes

### Run locally

```
$ git clone https://github.com/hunterloftis/cltjs-node.git
$ cd cltjs-node
$ npm install
$ DEBUG=health:* npm run dev
```

### Deploy to Heroku

```
$ heroku create
$ git push heroku master
```

### See each step as the app evolves

```
$ git checkout v1
$ git checkout v2
...
$ git checkout v9
```
